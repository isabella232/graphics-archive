Curriculum

  • Crafted by: Administrators
  • Scope: 1-year goals
  • Example: “Kindergarteners learn the number names as they count (to 100 by 1s and 10s) and write number names (from 0 to 20).” … “Students learn that each successive number name refers to a quantity that is one larger as they count objects and say the corresponding number names.”

Lesson Plan

  • Crafted by: Teachers
  • Scope: Week-by-week, day-by-day and minute-by-minute goals
  • Example: Specific class activities and assignments, like this Count ’n Match worksheet.
